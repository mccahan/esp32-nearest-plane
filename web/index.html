ESP32 Nearest Plane

ADS-B flight tracker for Guition ESP32-S3-4848S040

Install Firmware

Connect your ESP32-S3 board via USB and click the button below.

Your browser does not support Web Serial.
Use Chrome or Edge on desktop. Serial access is not allowed in this context.

Requires Chrome or Edge. The device will be erased during first install.

After Installation

  1. The device starts in AP mode — connect to the ESP32-Display WiFi network
  2. Open 192.168.4.1 in your browser to configure WiFi and location
  3. The device reboots and begins tracking nearby aircraft

What You Need

  1. Guition ESP32-S3-4848S040 (480x480 display)
  2. USB-C cable (data, not charge-only)
  3. WiFi network with internet access
